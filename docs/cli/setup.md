# `ozzo setup`

Initialize `~/.ozzobot/ozzo.json` and the agent workspace.

Related:
- Getting started: [Getting started](/start/getting-started)
- Wizard: [Onboarding](/start/onboarding)

## Examples

```bash
ozzo setup
ozzo setup --workspace ~/ozzo
```

To run the wizard via setup:

```bash
ozzo setup --wizard
```
