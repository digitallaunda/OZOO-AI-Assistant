# `ozzo tui`

Open the terminal UI connected to the Gateway.

Related:
- TUI guide: [TUI](/tui)

## Examples

```bash
ozzo tui
ozzo tui --url ws://127.0.0.1:18789 --token <token>
ozzo tui --session main --deliver
```
